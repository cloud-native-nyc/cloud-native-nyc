Co-Create Our Governance 🛠️

This governance model is an Open Source Draft.

Our Mission

Decentralize power & bring back the NYC Cloud Native Community.

We manage decisions transparently via GitHub to enforce open stewardship.

Vendor Neutrality Guardrail

  • Organizers elected by simple majority.
  • Majority vote must include an organizer from another company.
  • No single company can hold >50% of organizer seats.

👉 Open an issue on GitHub to debate rules live!

Open the floor. Explain the new "Anti-Capture Guardrail" (50% vendor neutrality rule and voting requirements). Ask the audience: "Do these incentives feel right to you? What is missing?"
